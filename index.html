Interested in a custom TabBot?

Right now TabBot only monitors the availability of the Xbox Series X and Playstation 5 in US stores, but it was built to easily handle any configuration for any site, anywhere based on a simple JSON object. You simply plug in your target website and a HTML selector and TabBot will refresh the page every 25 seconds and alert you when it exists on the page.

If users express interest, I will build out the backend infrastructure necessary to support custom TabBots - with the goal of eventually making TabBot accessible to non-technical users. Even if you are not familiar with HTML, feel free to sign up for updates on the product as I add things like custom alerts and free product tabs to the free chrome extension.



Current Roadmap
  • Release 1.0.0, a proof of concept which is set up to monitor PS5 / Xbox Series X availability
  • Release 2.0.0, allowing users to upload custom configuration files which enables monitoring for any given page for any custom selector
  • Add custom alert configuration (text message, email) and multi-profile configuration to TabBot.
  • Release 3.0.0, allowing non-technical users to configure / customize TabBot in a pure GUI configuration tool.
